# Contributing to Flux

Thank you for your interest in contributing to Flux!

## How to Contribute

- Please read the [README.md](./README.md) and [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) before contributing.
- Fork the repository and create your branch from `main`.
- Ensure your code matches the Flux philosophy: explicitness, safety, and no hidden magic.
- Write clear, maintainable, and well-documented code.
- Add tests for new features and bug fixes.
- Use conventional commits for all commit messages.
- Run all pre-commit and pre-push checks before submitting a pull request.

## Pull Request Process

- Ensure your PR passes all CI checks.
- Reference related issues in your PR description.
- Be responsive to feedback and update your PR as needed.

## Code of Conduct

Please note that this project is released with a [Code of Conduct](./CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
